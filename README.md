### Сайт для интернет-магазина. 

#### Настройка и запуск
1. Файл .env.template переименовать в .env и указать в нем параметры конфигурации Django. Ниже приведен пример:
```
DJANGO_LOGLEVEL=DEBUG
SECRET_KEY=YourDjangoSecretKey
DJANGO_DEBUG=0
DJANGO_ALLOWED_HOSTS=example.com,8.47.69.6
```
2. В папке nginx/nginx.conf в строке server_name example.com; заменить example.com на ваши доменные имена
3. Запуск из корня:
 docker-compose up
 
 При запуске контейнеров устанавливаются необходимые приложения, применяются миграции базы данных, а так же заполняется база данных из файла фикстур с тестовыми данными, необходимыми для демонстрации работы сервиса. Тестовые медиа файлы для объектов profile, product, category находятся в папке uploads. Пароль учетной записи "admin": "123"
