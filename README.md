# README.md

## 🚀 Быстрый старт

```bash
git clone <repository-url>
cd project
cp .env.template .env
```

### Настройка .env
```
DJANGO_LOGLEVEL=DEBUG
SECRET_KEY=YourDjangoSecretKey
DJANGO_DEBUG=0
DJANGO_ALLOWED_HOSTS=example.com,8.47.69.6
```

```bash
python manage.py migrate
python manage.py collectstatic
python manage.py createsuperuser
python manage.py runserver 0.0.0.0:8000
```

## 🎯 Структура сайта

```
📱 Главная страница
├── Избранные категории (3 шт.)
├── Топ товаров (8 популярных)
└── Слайдер Limited Edition (до 16)

🛒 Каталог товаров
├── Фильтры (название, цена, производитель, характеристики)
├── Сортировка (популярность, цена, отзывы, новизна)
├── Пагинация
└── Детальная страница товара + отзывы

🧾 Оформление заказа
├── Корзина
├── Параметры пользователя
├── Доставка (Доставка/Экспресс)
├── Оплата (Карта онлайн/при получении)
└── Подтверждение + оплата

👤 Личный кабинет (только авторизованные)
├── Сводка (профиль + последний заказ)
├── Профиль (ФИО, email, телефон, аватар ≤2MB)
└── История заказов (с повторной оплатой)

⚙️ Админка Django
├── Пользователи
├── Товары
├── Категории
├── Заказы
└── Отзывы
```

## 🎭 Роли пользователей

| Роль | Возможности |
|------|-------------|
| **Администратор** | Полный доступ к админке |
| **Покупатель** | Все публичные функции + ЛК |
| **Гость** | Каталог + корзина |

## 📋 Шапка и футер

**Шапка:**
```
[Соцсети] [Регистрация/Авторизация/ЛК] Logo [Сравнение|🛒] Меню|Категории|🔍Поиск
```

**Категории:** до 2 уровня вложенности (текст + иконка)

**Футер:** название + статичные страницы

## 🔧 Функционал каталога

**Список товаров:** название, фото, описание, цена, отзывы, кнопка "Купить"  
**Фильтры:** название, цена, производитель, характеристики (список/галочка/текст)  
**Сортировка:** популярность, цена, отзывы, новизна (▲/▼)

## 💳 Оформление заказа

```
1. Параметры (ФИО, email, телефон) → Авторизация/Регистрация
2. Доставка (Город, Адрес) → Доставка/Экспресс
3. Оплата → Онлайн/При получении
4. Подтверждение → Оплата
```
